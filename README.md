## Projeto de site feito em php com laravel e com sistema de crud


### Funcionalidades

- Possue sistema de cadastro de usuario
- Login de usuario
- Opção de criar um usuario
- Atualizar esse usuario
- Apagar esse usuario do banco de dados.

  #### Tecnologias

  - PHP
  - Laravel
  - MySQL
  - Apache
  - JavaScript
  - Sass
